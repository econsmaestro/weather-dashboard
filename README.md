# Weather Dashboard

A static, dependency-free weather dashboard: current conditions + 7-day forecast + city search. No backend, no API keys, no build step.

## Stack

Plain HTML / CSS / vanilla JS. Weather and geocoding data from [Open-Meteo](https://open-meteo.com/) (free, no API key required).

## Run locally

Just open `index.html` in a browser, or serve the folder with any static file server.

## Deploy (GitHub Pages)

1. Push this repo to GitHub.
2. Settings → Pages → Source: `main` branch, root.
3. Site will be live at `https://<username>.github.io/<repo>/`.
